# Assets Directory

This directory contains all static assets for the Pepe's Toys website.

## Recommended Structure

```
assets/
├── images/
│   ├── hero/
│   │   └── hero-background.jpg (1440x407px - Main hero background image)
│   ├── collections/
│   │   ├── voladeras-1.jpg (225x225px)
│   │   ├── voladeras-2.jpg (225x225px)
│   │   ├── voladeras-3.jpg (225x225px)
│   │   ├── voladeras-4.jpg (225x225px)
│   │   └── voladeras-5.jpg (225x225px)
│   └── logo/
│       └── pepestoys-logo.png (250x220px)
└── icons/
    └── bird-icon.svg (50x50px)
```

## Image Specifications

### Hero Section
- **Hero Background**: 1440x407px
  - Format: JPG
  - Subject: Colorful parrot/macaw
  - Current placeholder: Pexels stock image

### Logo
- **Pepe's Toys Logo**: 250x220px
  - Format: PNG with transparency
  - Current placeholder: Hummingbird image

### Collection Cards
- **Size**: 225x225px (1:1 aspect ratio)
- **Format**: JPG
- **Subject**: Bird cages, aviaries, bird toys, etc.
- **Note**: Images will have a 40% black overlay applied via CSS

## Color Palette

- **Primary Green**: `#008F24`
- **Dark Green**: `#005114`
- **Pink/Red**: `#EE193F`
- **Magenta**: `#A41262`
- **White**: `#FFFFFF`
- **Black Overlay**: `rgba(0, 0, 0, 0.6)` and `rgba(0, 0, 0, 0.4)`

## Fonts

- **Font Family**: Inter
- **Weights Used**: 300 (Light), 400 (Normal), 500 (Medium)
